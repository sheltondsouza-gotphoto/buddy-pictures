# Entagged Prototype

A React-based prototype application for subject tagging and management.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Build

Create a production build:

```bash
npm run build
```

## Deployment to GitHub Pages

See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed deployment instructions.

Quick deployment:

1. Update the `homepage` field in `package.json` with your GitHub Pages URL
2. Run `npm run deploy`

## Features

- Single and multi-tagging modes
- Subject filtering
- Active subject highlighting
- Maximum subject limit warnings
- Mode switching with confirmation modals

## Technologies

- React 19
- Tailwind CSS
- Create React App
